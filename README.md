# NotePal

一个简单的笔记管理工具

> Since this tool is intended for Chinese users, this documentation will be written in Chinese

> 此工具面向中文用户，因此这个仓库的文档均使用中文。

## 功能

- [x] 解析「微信读书」笔记
- [ ] 解析「读库」笔记
- [x] 导出笔记到 Readwise 支持的 CSV 格式
- [ ] 自定义导出格式
